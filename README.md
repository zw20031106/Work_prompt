# Work_prompt

这是一个用于整理和沉淀日常工作、科研写作、文献阅读、文档转换以及 AI Agent 使用经验的提示词与 skill 资源仓库。

仓库当前内容主要包括两部分：

- `常用提示词/`：可直接复制使用的高频提示词模板。
- `skill/`：收集、备份和整理的各类 Agent/LLM skill 资源包，便于后续迁移、二次整理和扩展。

本仓库会持续更新。后续会继续补充更多经过实际使用验证的提示词、工作流模板、科研辅助流程、文档处理方案和可复用 skill。

## 常用提示词

### Word 图片公式 OCR 转 Markdown

文件：`常用提示词/Word图片公式OCR转Markdown_通用提示词.md`

用于将 Word 文档中的图片、公式、题目、解答过程等内容尽可能完整地转换为 Markdown。适合处理教材、讲义、题库、课程资料、截图型公式和包含大量图片内容的 Word 文件。

### 学术论文深度阅读与分析

文件：`常用提示词/学术论文深度阅读与分析提示词.md`

用于辅助阅读和拆解学术论文，包括研究背景、核心问题、方法设计、实验结果、创新点、局限性和可借鉴思路等。适合文献精读、组会准备、课题调研和论文复现前的结构化分析。

### 期刊论文翻译与 LaTeX 重建

文件：`常用提示词/期刊论文翻译_LaTeX重建提示词.md`

用于将期刊论文进行较高保真的中文翻译与 LaTeX 结构重建。重点关注正文结构、图表说明、公式、引用、章节层级和可编译性，适合论文学习、翻译存档和 LaTeX 模板化整理。

## Skill 资源

`skill/` 目录下收集了多个 skill 或 Agent 工作流相关资源，主要用于：

- 学术论文写作、审稿、修改、回应审稿意见；
- 深度研究、文献检索、证据整理和系统综述；
- Nature 风格论文、图表、PPT、数据可用性声明等科研工作流；
- 演示文稿、文档处理、设计原型和自动化任务；
- Agent 行为约束、调试、TDD、计划执行和代码审查流程；
- 提示词反模式识别与输出质量控制。

这些资源目前以归档和备份为主，后续会逐步整理出更清晰的使用说明、推荐组合和实践案例。

## 使用方式

### 直接使用提示词

进入 `常用提示词/` 目录，打开对应 Markdown 文件，复制其中的提示词到 ChatGPT、Codex、Claude、Gemini 或其他大模型工具中，根据自己的任务替换输入材料即可。

建议使用时保留以下信息：

- 任务目标；
- 输入材料；
- 输出格式；
- 质量要求；
- 是否需要分步骤执行；
- 是否需要在最后进行自检。

### 整理为个人工作流

可以根据自己的使用场景，将提示词进一步拆分为：

- 文献阅读类；
- 论文写作类；
- 翻译润色类；
- 文档转换类；
- PPT/汇报类；
- Agent 行为规范类；
- 代码与项目管理类。

后续本仓库也会逐步按这些类别补充索引。

### 使用 skill 资源

`skill/` 下的内容适合有 Agent 使用经验的用户参考。不同 skill 的安装方式、触发方式和适配平台可能不同，使用前建议先阅读对应目录中的 `README.md`、`SKILL.md` 或相关说明文件。
