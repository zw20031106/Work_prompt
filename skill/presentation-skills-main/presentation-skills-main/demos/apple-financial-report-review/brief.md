# Apple Inc. FY2025 财报点评

## 任务定义
- 目标读者：具备财务、科技行业和二级市场基础知识的学术交流读者。
- 主使用场景：围绕 Apple Inc. FY2025 Form 10-K 做一份严谨正式的财报点评 PPT 初稿。
- 目标动作：帮助读者快速判断 FY2025 财务表现的主线、经营质量、业务结构变化和主要风险。
- 参考模板文件：无指定 PPT 模板。
- 模板 / 品牌约束：仿照中信风格的正式研报语气、低饱和配色、证据优先、页脚清晰；不使用中信、Apple 或第三方机构 logo，不复制受保护版式。
- 交付物要求：在本工作区交付可编辑 PPTX、数据底稿、10-K 原文、逐页预览图、基础质量检查结果和交付说明。
- 验证要求：运行数据采集脚本、业务拆分脚本、PPT 构建脚本、package preflight、structure precheck、LibreOffice 预览导出和 render review；检查预览图与关键数据一致。

## 模板取证
- 页面系统判断：没有参考 PPTX，采用空白页直生的自建研究报告页族。
- 关键母版 / layout 元素：脚本统一生成页眉、标题区、页脚、页码、数据来源短注和风险/免责声明页。
- 字号系统：中文 `宋体`，英文 `Times New Roman`；封面标题 28pt，正文页标题 18.5pt，正文小四 12pt，表格五号 10.5pt，页脚 6.3-6.6pt。
- 段落系统：正文首行缩进约 0.24 英寸，段前段后 6pt，1.5 倍行距；表格使用五号字、单倍行距、段前段后 0、无首行缩进，表头居中，index / 类目列居左，文本列居左，财报数值列靠右。
- 颜色系统：采用接近中信证券正式研报的深红、暗金、暖棕、浅米灰和灰线体系；不使用任何中信、Apple 或第三方机构 logo。
- 计划采用的构建路线：`python-pptx` 生成可编辑 PPT，原生 Office chart 和原生表格为主，少量 shape 作为结论卡片。
- 最小 PoC 结论：当前环境具备 `python-pptx`、LibreOffice 与 `pdftoppm`，可走可编辑生成和预览导出链路。

## 风格与边界
- 风格参考：正式卖方研报的结构、措辞和信息密度；标题直接写判断，图表服务证据，结论避免夸张化。
- 允许使用的素材：SEC 官方 API、SEC 10-K 原文、脚本生成的图表和表格、通用几何图形。
- 不允许发生的错误：不得编造未披露指标，不得给出投资评级或目标价，不得使用侵权 logo，不得把数据源不一致的问题静默处理。

## 数据口径
- 核心三表数据来自 SEC companyfacts API，单位转换为十亿美元或百分比后用于页面展示。
- 产品线、地区与 Products/Services 毛利率来自已下载的 FY2022 与 FY2025 10-K HTML 表格，覆盖 FY2021-FY2025。
- FY2025 指 Apple fiscal year ended September 27, 2025，对应 10-K filed October 31, 2025。

## 强制声明
- 风险声明页必须覆盖产品创新、竞争、供应链、监管、汇率、税务、市场估值和数据口径风险。
- 免责声明页必须包含“仅供学术交流使用，不代表任何组织机构，机构和个人的观点和立场”。
