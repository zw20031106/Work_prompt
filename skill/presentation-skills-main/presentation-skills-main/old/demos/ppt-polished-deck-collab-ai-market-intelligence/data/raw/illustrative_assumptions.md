# Illustrative Assumptions

**数据口径。** 所有数值均为 skill demo 所用的 fabricated data，单位统一为十亿美元或标准化评分，不引用真实研究机构口径。

**收入口径。** `market_layer_revenue.csv` 把 AI 市场拆成 `Compute + Models`、`Tools + Orchestration`、`Applications + Services` 三层，并保证总额从 2023 的 `155` 增长到 2028E 的 `890`。

**利润口径。** `profit_pool_mix.csv` 用百分比表示行业 EBIT pool 在不同层之间的分配，强调“利润向下游迁移”的结构变化，而不是给出真实财务预测。

**需求口径。** `enterprise_budget_rank.csv` 和 `sector_adoption_heatmap.csv` 分别描述真实付费 workflow 的优先级与行业 adoption intensity，用于证明“采用很广，付费很集中”。

**建议口径。** `strategy_archetype_scores.csv` 把不同 archetype 放到同一评分框架下比较，评分是 strategy tool，不是财务事实。
