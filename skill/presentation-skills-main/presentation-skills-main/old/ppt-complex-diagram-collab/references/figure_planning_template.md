## 4. 一致性约束
- 同一组件在不同图中的命名必须一致。
- 同一数据实体（frame/snapshot/run）路径命名必须一致。
- 主图和子图之间的编号与引用关系必须可追溯。
- 主图（Fig00）先保证 3 秒可读（层级清晰、edge budget 受控），再考虑信息密度。

- `*.pptx`：论文/汇报可操作图源。
- `style_spec.md`（建议）：调色盘、字体、边预算、箭头/连线语义约束。
