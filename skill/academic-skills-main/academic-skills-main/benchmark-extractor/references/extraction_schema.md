# 抽取 Schema

使用下列字段做统一抽取。未明确写出的内容一律写 `未知`。

| 字段 | 含义 | 填写要求 |
| --- | --- | --- |
| 论文标题 | 论文名称 | 保留原题目 |
| 任务 | 论文关注的任务 | 如多任务，逗号分隔 |
| 数据集 | 使用的数据集或 benchmark | 未明确写出则填 `未知` |
| 指标 | 主要评价指标 | 如有多个指标，按论文主叙事排列 |
| baseline | 对比方法 | 写主要 baseline，不要无限展开 |
| 是否报告 SOTA | 是否明确声称达到或刷新 SOTA | `是` / `否` / `未明确` |
| 是否开源代码 | 代码是否公开 | `是` / `否` / `未知` |
| 是否开源数据 | 数据是否公开 | `是` / `否` / `未知` |
| 评测设置说明 | 评测设定、拆分方式、零样本或少样本等 | 保持简洁 |
| 备注 | 其他需要提醒的事项 | 如字段歧义、摘要信息不足 |

## 填写原则

- 只抽取论文明确给出的信息。
- 对摘要输入，优先保守，不做深层推断。
- 如果一篇论文覆盖多个任务或多个 benchmark，只保留与用户目标最相关的内容，或在备注中提示“还有其他设置未展开”。
