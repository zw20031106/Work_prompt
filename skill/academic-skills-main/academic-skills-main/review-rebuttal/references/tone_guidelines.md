# 语气指南

## 推荐语气

- 专业
- 克制
- 具体
- 非对抗

## 推荐表达

- “感谢审稿人指出这一点，我们同意该问题需要更清楚地说明……”
- “我们理解该 concern 的来源，并补充解释如下……”
- “这一点确实是当前版本的限制，我们将在 rebuttal / camera-ready 中明确说明……”
- “为回应这一问题，我们计划补充……；在当前输入范围内，我们暂不声称已有结果。”

## 避免表达

- “审稿人误解了我们的工作”
- “这个问题其实不重要”
- “显然”
- “我们的方法全面优于已有方法”

## 改写原则

- 把情绪性防御改写为信息性澄清。
- 把绝对化表述改写为基于证据的限定表达。
- 把无法支持的承诺改写为可执行的补充计划。
