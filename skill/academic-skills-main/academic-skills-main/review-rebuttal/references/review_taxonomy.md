# Review Concern 分类法

## 常见类型

- `novelty`：创新性不足、增量不明显
- `technical_clarity`：方法描述不清、关键设定缺失
- `experimental_support`：实验不充分、缺少 baseline / 消融 / 错误分析
- `evaluation_fairness`：比较不公平、设置不一致、指标不匹配
- `significance`：问题重要性或实际价值不足
- `positioning`：与相关工作关系不清、文献覆盖不足
- `limitation_scope`：适用范围、边界条件或失败案例不清
- `writing_structure`：写作、组织、图表可读性问题

## 严重性判断

- 高：直接影响接收判断，通常需要实验或关键澄清
- 中：不一定决定接收，但会显著影响 reviewer 信心
- 低：写作或表达层面的修订项

## 动作建议

- `必须补实验`
  适用于 experimental support、evaluation fairness 中的硬缺口
- `优先澄清`
  适用于技术细节不清、相关工作定位不清
- `承认局限`
  适用于短期内无法补齐、但需要诚实说明的边界问题
